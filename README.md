## Library yang Digunakan
```
- Numpy
- Pandas
- Matplotlib
- Seaborn
- Streamlit
```

## Setup Environment
```
pip install -r requirements.txt
```

## Run steamlit app
```
cd dashboard
streamlit run dashboard.py
```
